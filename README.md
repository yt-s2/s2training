# SingleStore Workshop
SingleStore Workshopで使用するNotebookの使用方法を解説します。

1. `getting-started-with-singlestore-jp.ipynb`をローカルPCにダウンロードします。
2. SingleStore Heliosに自身のアカウントでログインします。Heliosのアカウントを未作成の方は<a href=https://qiita.com/gootake/items/7e1ded389f90332fc6d1 target=new>こちらの記事</a>を参考にアカウントを作成してください。
3. 左側メニューから**Editor**に移動します。
4. 画面右上の**+New**をクリックし、**Import From File**を選択します。
5. 先ほどダウンロードしたNotebookファイルを画面上の**Drag file here**と書かれたグレー色の領域にドラッグし、Locationとして**Personal**を選択し、**Import**をクリックします。
6. SingleStore Workshop用のNotebookが開きます。

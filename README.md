# SingleStore Workshop
SingleStore Workshopで使用するNotebookの使用方法をご案内します。

1. `getting-started-with-singlestore-jp.ipynb`をローカルPCにダウンロードします。

   a. この説明の上部のファイル一覧画面にある`getting-started-with-singlestore-jp.ipynb`をクリックして開きます。
   
   b. 画面右上のDownloadアイコンをクリックしてファイルとしてダウンロードします。
<img src=images/download_raw_file.png>

3. <a href=https://portal.singlestore.com target=new>SingleStore Helios</a>に自身のアカウントでログインします。Heliosのアカウントを未作成の方は<a href=https://qiita.com/gootake/items/7e1ded389f90332fc6d1 target=new>こちらの記事</a>を参考にアカウントを作成してください。

4. ログイン後、左側メニューから**Editor**に移動します。
<img src=images/editor.png>

5. 画面右上の **+New**をクリックし、**Import From File**を選択します。
<img src=images/import_from_file1.png>

6. 先ほどダウンロードしたNotebookファイルを画面上の**Drag file here**と書かれたグレー色の領域にドラッグし、Locationとして**Personal**を選択し、**Import**をクリックします。
<img src=images/import_from_file2.png width=30%>

7. SingleStore Workshop用のNotebookが開きます。
<img src=images/s2_workshop_notebook.png>

Notebookのガイドに従ってハンズオンワークショップを進めてください。
